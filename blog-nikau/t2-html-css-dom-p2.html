Udacity HTML and CSS lessons

The Udacity HTML and CSS lessons covered:

  • grid-based design
  • building and applying frameworks such as Bootstrap
  • responsive web pages
  • semantic structure
  • refinement

Grid-based design refers to a structure for web page layouts made up of rows and columns.

Understanding the way a page is divided into grids will help you build a framework; a collection of classes that will make page layouts easier to implement.

Believing the hype on responsive design

Recognising that people access information via a range of different devices is more than hype, it's smart.

Implementing practices that create more responsive web pages is even smarter. The Udacity lessons offer suggestions to make a web page responsive:

  • using percentages instead of fixed pixels to define column size
  • adding flexible box styles and overflow
  • add media specific css e.g. @media to change css properties depending on device screen size
  • and CSS resetting using mormalize.css.

Applying the framework

Semantic structure consists of semantic tags. Semantic tags give meaning to web page content that both computers and humans understand.

The semantic structure enables humans to understand the meaning of the content and structure For example, nav and span tags offer no semantic meaning to content. However, the following semantic tags add structural meaning (semantics):

  • heading - h1, h2, h3, h4...
  • image - img
  • a form - form

The lessons demonstrated fine tuning methods (i.e. code, test, refine), continuos improvement tips and links to tools such as Google Fonts, placeholder text sites and Bootstrap.

The additional lesson covering Bootstrap helped to reinforce how divs, rows, columns and classes can work.
